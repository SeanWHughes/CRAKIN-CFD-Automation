# CRAKIN CFD Automation
CRA:KIN is an automated workflow to help researchers efficiently perform massive computational fluid dynamics (CFD) parameter optimization projects (or any other format of CFD design of experiments (DOE) work). 

It functions via connecting PTC Creo® Parametric --> ANSYS Fluent® --> OpenFOAM along with Excel sheets throughout the whole pipeline. The decision to base the CRA:KIN architecture on these software was entirely for their notable accessibility to researchers (i.e. free or low-cost).

The codebase works using a handful of key Excel spreadsheets that the user pre-configures according to their particular case setup.

In the future, I hope to expand CRA:KIN to allow for integration with other commonly used intelligent parametric CAD software (e.g. SOLIDWORKS®) and especially, expand it for use with a viable opensource meshing software.
